# Smart Civic Issue Reporting Platform

This project is developed under the Open Innovation category.

## Problem Statement
Urban civic issues like potholes, garbage overflow, water leakage,
and streetlight failures often face delayed resolution.

## Proposed Solution
A smart digital platform where citizens can report civic issues
using images, GPS location, and real-time tracking.

## Key Features
- Issue reporting with image upload
- Auto location detection
- Real-time status tracking
- Admin dashboard

## Technologies Used
- Google Maps API
- Firebase
- Google Cloud Platform
